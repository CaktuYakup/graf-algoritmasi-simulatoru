# Graf Algoritması Simülatörü
Bu proje, Graf Teorisindeki temel algoritmaların web tarayıcısı üzerinden görselleştirilmesini sağlayan uygulamadır.

## Temel Özellikler
  Arayüz üzerinden sürükle-bırak yöntemiyle serbest düğüm ve kenar oluşturma.
  Tek yönlü, çift yönlü ve ağırlıklı kenar desteği.
  Algoritmaların çalışma prensiplerinin adım adım canlı olarak izlenebilmesi.
  Negatif döngü ve ulaşılamayan kopuk graf (disconnected graph) gibi istisnai durumların sistem tarafından tespit edilmesi.

## Desteklenen Algoritmalar
  Djikstra, Bellman-Ford, Prim, Kruskal

## Kurulum ve Kullanım
  Uygulama herhangi bir sunucu altyapısı, paket yöneticisi veya derleyici gerektirmez.
  Projeyi bilgisayarınıza indirip index.html dosyasını tarayıcıda çalıştırın
